# context
context: 通过提供自由的模块，简洁的接口，动态的结构，让编程变得更简单，让世界变得更美好。

## 数据结构
* ARM: 寻址与指令
* Linux: 文件与进程
* HTTP: 表示与会话

## 开发流程
* 设计: 协议与流程
* 编程: 接口与框架
* 测试: 语句与表达式

* 功能树: Caches Configs Commands
